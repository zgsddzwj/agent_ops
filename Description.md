# AgentOps - 智能体运维平台

## 项目概述

AgentOps是一个面向LangChain/LangGraph生态的全栈智能体运维(MLOps for AI Agents)平台，旨在为AI Agent项目提供统一的可观测性、评估、监控和安全保障基础设施。

**一句话描述：** 专为LangChain/LangGraph智能体设计的一站式运维平台，提供评估、监控、安全评测和多模型对比能力。

## 核心特性

### 🔧 多项目统一管理
- 支持任意外部AI项目(`ai_project`)的注册和接入
- 项目级Dashboard隔离展示
- 统一的CLI入口和API管理

### 📊 全面评估体系
- **离线评估框架**：支持smoke test、regression测试套件
- **多维度判定**：精确匹配和行为逻辑判定
- **内置数据集**：提供开箱即用的评估数据集

### 🔍 全链路可观测性
- **LangChain Callback集成**：自动采集LLM调用、Tool执行、Chain流转的全过程Trace
- **实时性能监控**：TTFT(首Token时间)、E2E延迟、Token消耗量
- **成本精确核算**：支持P50/P95分位统计和USD成本计算

### 🤖 多模型对比能力
- **模型性能对比**：同一项目下横向对比OpenAI/Qwen/DeepSeek等主流模型
- **预设配置包**：提供domestic、international、sota、cost_efficient等多种预设组合
- **详细对比矩阵**：TTFT、E2E延迟、成本、质量等多维度指标对比

### 🛡️ 安全防护体系
- **安全评测套件**：提示词注入、Jailbreak攻击、数据泄露检测
- **运行时防护**：SDK集成SecurityPipeline，提供输入过滤、注入检测、PII脱敏、限流
- **测试用例库**：内置多维安全测试用例和扫描工具

### 🚨 智能告警系统
- **多维度阈值告警**：成本、延迟、错误率、安全通过率等指标监控
- **灵活通知机制**：支持Webhook推送
- **自定义告警规则**：针对不同业务场景定制化告警策略

## 技术架构

### 分层架构设计
```
┌─────────────────┐     SDK (实时)      ┌──────────────────┐
│  ai_project A   │ ──────────────────► │                  │
├─────────────────┤                     │  FastAPI Backend │
│  ai_project B   │     CLI (离线)      │  PostgreSQL      │
└─────────────────┘ ──────────────────► │  Redis + Worker  │
                                        └────────┬─────────┘
                                                 │
                                        ┌────────▼─────────┐
                                        │  Next.js Dashboard│
                                        └──────────────────┘
```

### 技术栈组成
- **SDK层**：Python instrumentation + SecurityPipeline
- **CLI工具**：基于Typer的命令行接口，支持init/link/eval/benchmark/security/check
- **后端服务**：FastAPI + SQLAlchemy + Alembic，异步架构
- **数据存储**：PostgreSQL主存储 + Redis缓存和队列
- **任务处理**：ARQ异步任务框架，支持评估/基准测试/安全扫描等后台任务
- **前端Dashboard**：Next.js 14构建的现代化管理界面

## 工作流程

### 接入流程
1. **初始化项目**：`agent-ops init` 生成配置文件
2. **注册项目**：`agent-ops link` 获取API Key
3. **集成SDK**：在AI项目中引入AgentOps回调处理器
4. **开始监控**：实时采集执行数据和性能指标

### 评估流程
1. **配置评估集**：定义评测用例和预期结果
2. **运行评估**：`agent-ops eval run` 执行测试套件
3. **查看结果**：Dashboard查看评估报告和质量指标

### 安全扫描流程
1. **配置安全策略**：设置检测规则和阈值
2. **执行扫描**：`agent-ops security scan` 运行安全测试
3. **处理告警**：根据扫描结果调整安全策略
